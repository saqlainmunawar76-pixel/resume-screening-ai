🚀 Task 2 Complete — AI-Powered Resume Screening & Candidate Ranking System | TEYZIX CORE Internship

Just wrapped up my second AI task as part of the TEYZIX CORE Internship Program (June Batch 2026)! 🎯

The challenge: build a system that helps recruiters automatically screen and rank resumes against job descriptions — something real HR teams struggle with daily when reviewing hundreds of applications.

🔧 What I built:
✅ PDF resume parser that extracts name, contact info, skills, education, experience, and certifications automatically
✅ Job description analyzer that separates required vs. preferred skills and experience requirements
✅ An explainable matching engine — every candidate gets a transparent score breakdown (skill match + semantic similarity + experience), not just a black-box number
✅ 7/7 bonus features: AI interview question generation, skill-gap analysis, an AI chat assistant for recruiters, resume improvement suggestions, candidate clustering, semantic vector search, and multi-job candidate matching

💡 Key learnings:
- Designing scoring systems that are explainable, not just accurate — recruiters need to know WHY a candidate ranked where they did
- Building AI features with graceful fallback so the system stays usable even without an API key
- Combining classic NLP techniques (TF-IDF, regex) with generative AI (Gemini) for a system that's both reliable and intelligent

Built with Python, Streamlit, scikit-learn, ChromaDB, and Google Gemini API.

This project reflects exactly the kind of AI automation work I want to keep building toward — practical systems that solve real business problems.
